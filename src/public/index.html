Time-Shift Radio

Access radio streams with time-shifting capabilities

About the Service

Time-Shift Radio allows you to listen to radio streams with a time delay, giving you the ability to pause, rewind, and catch up on content you might have missed.

API Endpoints

GET /health

Check the health status of the service.

GET /stats

Get detailed statistics about the buffer and system status.

GET /stream

Access the time-shifted stream (coming soon).

Current Status

Fetch the current service status from the health endpoint or check detailed statistics.
